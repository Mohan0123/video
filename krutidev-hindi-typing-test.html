About Kruti Dev 010 and Hindi Typing

In the digital landscape of India, Hindi typing remains a critical skill for administrative and government roles. While Unicode (Mangal) is gaining popularity, Kruti Dev 010 remains the standard for many state-level examinations and legacy data entry systems.

Why is Kruti Dev Still Relevant?

Kruti Dev is a non-Unicode font that uses the Remington typewriter keyboard layout. Unlike Inscript (used by Unicode/Mangal), the Remington layout is often preferred by typists who transitioned from physical typewriters. Many state public service commissions in Bihar, Uttar Pradesh, Madhya Pradesh, and Rajasthan still mandate or offer Kruti Dev as an option for their typing skill tests.

How Speed is Calculated

We use standard typing test formulas used in government exams:

  • Gross Speed: Total Characters Typed / 5 / Time (min).
  • Net Speed: (Total Characters - Errors) / 5 / Time (min).
  • Accuracy: Percentage of correct characters against total characters typed.

Frequently Asked Questions (FAQ)

1. Do I need to install Kruti Dev font to use this test?

No, this tool uses web standard fonts to display Hindi. However, if you have Kruti Dev 010 installed on your computer, the text area will prioritize that font for a native experience. If not, it works perfectly with standard Hindi display.

2. Is this tool free to use?

Yes, this Kruti Dev Typing Test is 100% free and open for unlimited practice sessions.

3. How is WPM calculated?

We assume an average word length of 5 characters. Your Gross WPM is the total characters typed divided by 5, divided by the time elapsed.

4. Can I use this on mobile?

Yes, the website is fully responsive. However, for the best typing speed results, we recommend using a physical keyboard connected to a desktop or laptop.

5. Which exams require Kruti Dev typing?

Exams like MP CPCT, UP Police Computer Operator, Rajasthan High Court LDC, and various SSC clerical exams often use or allow Kruti Dev (Remington layout).
