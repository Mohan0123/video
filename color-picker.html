Understanding Color Formats

HEX (Hexadecimal): The most common format for web design, starting with a hashtag (#) followed by six characters (e.g., #FFFFFF for white).

RGB (Red, Green, Blue): Defines colors by the intensity of red, green, and blue light. Used widely in screen displays and CSS (e.g., rgb(255, 255, 255)).

HSL (Hue, Saturation, Lightness): A more intuitive way to describe color, often used by designers to adjust shades without changing the fundamental color.

What is the difference between HEX and RGB?
They represent the same color but in different formats. HEX is a base-16 code used primarily in HTML/CSS, while RGB uses decimal numbers (0-255) to represent Red, Green, and Blue channels.
